jQuery Slug Suggest

Basic Example

Here we're selecting an input where name="title".

$('input[name=title]').slugular();

Combining Multiple Fields

You can combine multiple source fields into one slug field. Just provide a selector that matches all of the relevant source fields.

Note here we're also using the slugSelector option to override which field the slug is inserted into. Usually this isn't necessary, but it's handy when you have multiple slug fields on one page, or just want to use a custom name. 

$('input[name=first_name],input[name=last_name]').slugular({ slugSelector: 'input[name=name_slug]' });

Other Options

$('#my_title_field').slugular({ slugSelector: '#my_slug_field', separator: '-', suggestionClass: 'my_custom_class' });
